# mssql

## Usage

```yaml
source:
  name: mssql
  config:
    connection_url: sqlserver://admin:pass123@localhost:3306/
```

## Inputs

| Key | Value | Example | Description |    |
| :-- | :---- | :------ | :---------- | :- |
| `identifier` | `string` | `my-mssql` | Instance alias, the value will be used as part of the urn component | *required* |
