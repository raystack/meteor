# shield

## Usage

```yaml
source:
  type: shield
  config:
    host: shield.com:80
```

## Inputs

| Key | Value | Example | Description |    |
| :-- | :---- | :------ | :---------- | :- |
| `host` | `string` | `shield.com:80` | Shield' GRPC host | *required* |
